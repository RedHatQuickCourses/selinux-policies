IN RHEL9 WE SHIP THE TARGETED SELINUX POLICY BY DEFAULT

SELINUX ALLOW RULE SYNTAX WITH TYPES

TYPE_TRANSITION TYPE1 TYPE2:OBJECT_CLASS NEW_TYPE;
